Introduction

Welcome to the ViaAds - ViaBill API documentation.

Production Url: https://viabill.viaads.dk
Development Url: https://dev.viabill.viaads.dk

Authentication

To authenticate your requests to the ViaAds API, you will need to include your API key in the header of each request.

ApiKey: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

You must replace xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx with your personal API key.

Discounts Section

Discounts

curl -X 'GET' \
  'https://viabill.viaads.dk/discounts/<email>/<size>/<page>' \
  -H 'accept: text/plain' \
  -H 'ApiKey: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx'

JSON response example:

{
  "lastPage": 0,
  "discounts": [
    {
      "id": 0,
      "src": "string",
      "title": "string",
      "subtitle": "string"
    }
  ]
}

This endpoint is used to retrieve a list of discounts based on the Customers Preferences

HTTP Request

GET https://viabill.viaads.dk/discounts/<email>/<size>/<page>

URL Parameters

Parameter Type Description
Email string The email of the customer requesting
Size int How many elements to recevice pr. page
Page int The page number


DiscountPagination Properties

Property Type Description
LastPage integer The last page of the pagination.
Discounts List<DiscountRef> The list of discount elements.

DiscountRef Properties

Property Type Description
Id long The ID of the discount.
Src string The source of the discount.
Title string The title of the discount.
Subtitle string The subtitle of the discount.

Discount

This endpoint is used to retrieve a discount incl. details by its id.

curl -X 'GET' \
  'https://viabill.viaads.dk/discounts/<discountId>/<email>' \
  -H 'accept: text/plain' \
  -H 'ApiKey: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx'

JSON response example:

{
  "id": 0,
  "overviewImageSrc": "string",
  "bannerImageSrc": "string",
  "title": "string",
  "subtitle": "string",
  "description": "string",
  "expireDate": "datetime",
  "discountCode": "string",
  "redirectLink": "string",
  "discountType": "string",
  "comingSoon": false,
  "isSubscribed": false
}

HTTP Request

GET https://viabill.viaads.dk/discounts/<discountId>/<email>

URL Parameters

Parameter Type Description
DiscountId long The ID of the discount
Email string The email of the customer requesting


Discount Properties

Property Type Description
Id long The identifier of the coupon.
OverviewImageSrc string The image source for the overview image of the coupon.
BannerImageSrc string The image source for the banner image of the coupon.
Title string The title of the coupon.
Subtitle string The subtitle of the coupon.
Description string The description of the coupon.
ExpireDate DateTime The expiration date of the coupon.
DiscountCode string The discount code associated with the current customer.
RedirectLink string The redirect link associated with the coupon.
DiscountType string The type of Discount (Coupon or Generic Webshop Discount).
ComingSoon bool Whether or not the Discount is a Coming Soon Discount.
IsSubscribed bool Whether or not the Customer is subscribed to a Coming Soon Discount.

Event

This endpoint is used for pushing discounts events, to track customer behavior
Event Types:
- discount_details_viewed
- discount_link_opened
- discount_code_copied 

curl -X 'POST' \
  'https://viabill.viaads.dk/discounts/event' \
  -H 'accept: text/plain' \
  -H 'ApiKey: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx' \
  -H 'Content-Type: application/json' \
  -d '{
  "discountId": 0,
  "customerEmail": "string",
  "eventType": "string",
  "eventDate": "2023-06-14T10:35:34.377Z"
}'

JSON response example:

{
  "status": 200,
  "message": "Success"
}

HTTP Request

POST https://viabill.viaads.dk/discounts/event
