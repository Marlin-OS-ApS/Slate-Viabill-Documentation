    Introduction

    Welcome to the ViaAds API documentation.

    Authentication

    To authenticate your requests to the ViaAds API, you will need to include your API key in the header of each request.

    ApiKey: key123

    You must replace key123 with your personal API key.

    Coupons

    Coupons

    curl -X 'GET' \
  'https://coupons.viaads.dk/Coupons/<debtorAccountId>/<size>/<page>' \
  -H 'accept: text/plain' \
  -H 'ApiKey: key123'

    JSON response example:

    {
  "lastPage": 0,
  "coupons": [
    {
      "id": 0,
      "src": "string",
      "title": "string",
      "subtitle": "string"
    }
  ]
}

    This endpoint is used to retrieve a list of coupons

    HTTP Request

    GET https://coupons.viaads.dk/Coupons/<debtorAccountId>/<size>/<page>

    URL Parameters

    Parameter Description
    DebtorAccountId The ID of the kitten to retrieve
    Size The ID of the kitten to retrieve
    Page The ID of the kitten to retrieve

    This endpoint is used to retrieve a coupon incl. details by its id.

    HTTP Request

    POST https://coupons.viaads.dk/Coupons/<couponId>/<debtorAccountId>

    URL Parameters

    Parameter Description
    CouponId The ID of the kitten to retrieve
    DebtorAccountId The ID of the kitten to retrieve

    Event

    curl -X 'POST' \
  'https://coupons.viaads.dk/Coupons/<couponId>/<debtorAccountId>' \
  -H 'accept: text/plain' \
  -H 'ApiKey: test' \
  -H 'Content-Type: application/json' \
  -d '{
  "couponId": 0,
  "customerEmail": "string",
  "eventType": "string",
  "eventDate": "2023-06-14T10:35:34.377Z"
}'

    JSON response example:

    {
  "status": 200,
  "message": "Success"
}

    This endpoint is used for pushing coupons events, to track customer behavior

    HTTP Request

    POST https://coupons.viaads.dk/Coupons/event
